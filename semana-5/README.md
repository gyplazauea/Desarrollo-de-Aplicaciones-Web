# Galería Interactiva con JavaScript

Proyecto desarrollado como parte de la **Semana 5**.
Permite agregar imágenes mediante URL, seleccionarlas y eliminarlas dinámicamente utilizando JavaScript y manipulación del DOM.

## Tecnologías
- HTML
- CSS
- JavaScript

## Funcionalidades
- Agregar imágenes desde una URL
- Seleccionar una imagen
- Eliminar imagen seleccionada
- Publicación en GitHub Pages
